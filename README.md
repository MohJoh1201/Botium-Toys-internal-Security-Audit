# Botium Toys Internal Security Audit

This project contains an internal security audit for Botium Toys based on the NIST Cybersecurity Framework (NIST CSF).

## Project Overview
Botium Toys is a growing company that needs to improve its security posture and compliance with industry regulations.

## Documents Included
- Controls and Compliance Checklist
- Scope, Goals, and Risk Assessment Report

## Key Findings
- No encryption used for sensitive customer and credit card data
- No backups or disaster recovery plan in place
- No intrusion detection system (IDS)
- Missing least privilege and separation of duties controls

## Recommendations
- Implement encryption for data at rest and in transit
- Add backups and create disaster recovery plans
- Install IDS and improve monitoring
- Apply least privilege and separation of duties
- Use a centralized password management system
